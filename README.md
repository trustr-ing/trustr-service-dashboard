# Trustr Service Dashboard

This is a web based Nostr client allowing users to interact with the Trustr orchestrator. It provides the following user-friendly interfaces:

- User login via browser extension or remote signer interfaces
- Subscription management for logged in users 
- Configuration of new service requests and request templates.
- Publishing and deleting of service requests events
- Chaining service requests, the output events of one service request becomes input for another (with the `pov` config being a naddr address and `type` config matching the result tags in the output event)
- Monitoring the status of and results generated by those requests
